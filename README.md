## Installation

Create the environment and install dependencies

```sh
conda create env --name py37 python=3.7
pip install dival==0.6.1 pytorch==1.7 scikit-image==0.18.1
# install odl from source
```

Install the package in editable mode

```sh
pip install -e .
```
